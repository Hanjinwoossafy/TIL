### 계산기1

- 문자열로 된 계산식이 주어질 때, 스택을 이용하여 계산식의 값 계산 가능
- 문자열 수식 계산의 일반적인 방법
  - step1. 중위 표기법의 수식을 후위 표기법으로 변경한다.(스택이용)
  - step2. 후위 표기법의 수식을 스택을 이용하여 계산

```md
**중위표기법(infix notation)**
- 연산자를 피연산자의 가운데 표기하는 방법
    예) A+B
**후위표기법(profix notation)**
- 연산자를 피연산자 뒤에 표기하는 방법
    예) AB+
```
